## Docker images

### Penpot

Penpot is an open source design and prototyping platform.
[https://help.penpot.app/technical-guide/getting-started/docker/](https://help.penpot.app/technical-guide/getting-started/docker/)
```bash
docker compose -p penpot -f penpot-compose.yaml up -d
podman compose -p penpot -f penpot-compose.yaml up -d
```
